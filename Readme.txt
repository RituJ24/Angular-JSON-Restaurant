This project is made using Angular and Node (JSON Server)
Project is on Restaurant POS
The user should be an Admin

How to run this project?
1. Download and extract the zip file
2. Open the folder in vs code and cd to RestaurantApp
3. npm install + (app.module.ts's libraries)
4. Terminal 1: json-server --watch db.json
5. Terminal 2: ng serve -o OR ng serve
